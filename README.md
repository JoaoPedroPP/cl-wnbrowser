# README #

This is CL-WNBROWSER pre-alpha, version 0.

### How do I get set up? ###

* You'll need SOLR (see CONSTANTS.LISP for configuration options);
* You'll need SBCL or CCL;
* For easier/faster setup, quicklisp is highly recommended.
