API access to OWN-PT

Three URLs in this site also serve back JSON documents instead of HTML if you pass the header Accept: application/json. They are:
  • Document search: http://wnpt.brlcloud.com/wn/search
  • Synset page: http://wnpt.brlcloud.com/wn/synset
  • Activity search: http://wnpt.brlcloud.com/wn/search-activities
All the parameters accepted via browser are also recognized via web service call. Examples: 
curl -H "Accept: application/json" http://wnpt.brlcloud.com/wn/search?term=carro

curl -H "Accept: application/json" http://wnpt.brlcloud.com/wn/synset?id=03538300-n
