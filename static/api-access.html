API access to OWN-PT

Three URLs in this site also serve back s-expressions instead of HTML if you pass parameter sexp=yes. They are:
  • Document search: http://wn.mybluemix.net/search
  • Synset page: http://wn.mybluemix.net/synset
  • Activity search: http://wn.mybluemix.net/search-activities
All the parameters accepted via browser are also recognized via web service call. Examples: 
curl "http://wn.mybluemix.net/search?term=carro&sexp=yes"

curl "http://wn.mybluemix.net/synset?id=03538300-n&sexp=yes"
